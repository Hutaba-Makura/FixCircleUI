# Fix Circle UI - コミケWebカタログのUIを改善
コミックマーケットのWebカタログのUIを改善する拡張機能です。

## 機能

- お気に入りサークルリストやサークル検索で、選択した色のみ表示できます。
（例）レッド、ブルーのサークルのみを表示する　など

- サークルリストの外部サービスアイコン（Twitter, Pixivなど）をクリックした時ちゃんと外部サービスに飛べるようにしました。

- 会場MAPにおいても選択した色のみ表示する機能を実装しました。

## インストール方法

1. このリポジトリをクローンまたはZIPダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択
